# mbf_gridmap Navigation Stack

This repository contains the implementation of a navigation stack
using [grid_map](https://github.com/ANYbotics/grid_map) as the underlying data structure to represent the robot
environment. The global architecture is based on [move_base_flex](https://github.com/naturerobots/move_base_flex) and
can be compared, in some respects, to [mesh_navigation](https://github.com/naturerobots/mesh_navigation). The grid_map
is filled by [elevation_mapping_cupy](https://github.com/leggedrobotics/elevation_mapping_cupy).
